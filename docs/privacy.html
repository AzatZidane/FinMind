1. Кто мы

FinMind — персональное офлайн‑приложение для учёта финансов. Контакты: ismagilovazat48@gmail.com.

2. Какие данные обрабатываются

  • Введённые пользователем данные: доходы, расходы, категории, цели и долги; параметры профиля (аватар, никнейм, тема/валюта); по желанию — «сбережения» (фиат/криптовалюта/металлы).
  • Резервные копии и экспорт/импорт (JSON) — по инициативе пользователя.
  • Технические данные соединения (например, IP‑адрес, время запроса) могут кратковременно обрабатываться при работе онлайн‑функций и не используются для идентификации.
  • ИИ‑советник: текст запроса обрабатывается через сервер‑прокси FinMind с использованием технологий OpenAI.

3. Цели обработки

  • функционирование приложения;
  • опциональная синхронизация через iCloud (если включена);
  • поддержка и отладка (если пользователь вручную отправляет отчёт об ошибке).

4. Хранение данных

Данные бюджета и настроек хранятся локально на устройстве. При включении iCloud синхронизация регулируется правилами Apple. Запросы к ИИ обрабатываются через прокси FinMind и поставщика ИИ.

5. Передача третьим лицам

OpenAI (через прокси FinMind) — для обработки ИИ‑запросов; Apple (iCloud) — если включена синхронизация. Данные не продаются и не используются для таргетированной рекламы.

6. Права пользователя

  • экспорт/импорт (JSON);
  • удаление всех данных: «Настройки → О приложении → Удалить все данные…»;
  • отключение iCloud‑синхронизации;
  • обращение к разработчику: ismagilovazat48@gmail.com.

7. Безопасность

Передача данных по сети защищена (HTTPS). Мы принимаем меры для защиты данных, однако абсолютная безопасность не гарантируется.
